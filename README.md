# AROIMAK Ordering System

A tablet-based ordering system designed for a Thai restaurant.

## Features
- Thai audio pronunciation
- Customizable spice levels
- Interactive user interface

## Tech Stack
- PHP
- MySQL (XAMPP)
- HTML, CSS, JavaScript

## Purpose
This project was developed as a capstone project to improve ordering efficiency and user experience in restaurants.
